What is a typescript?
What advantages does it have?

  • Simplifies support and debugging
  • Offers advanced features such as interfaces, classes, and decorators
  • Helps write more reliable and scalable code

TypeScript can be used in various development scenarios
Code example 

						interface Rectangle {
							width: number;
							height: number;
						}
						
						function getArea(rectangle: Rectangle) {
							return rectangle.width * rectangle.height;
						}
						
						const myRectangle = { width: 10, height: 5 };
						console.log(getArea(myRectangle));  // Output: 50
